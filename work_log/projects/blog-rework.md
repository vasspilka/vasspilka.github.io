# Blog Publisher Project

## Why are we doing this?

We want a more modern blog with tools to help us publish articles to wider audiences.

### How to measure success

 * Easiness of making blog posts and changes
 * Load time for blog
 * Time to publish to Substack & Medium

## Problem Statement

### High Level

This project has gotten quite outdated. We want to revitalize it and include tools to help with publishing blog posts to various platforms like Medium and Substack.

The goal is to first streamline the publishing process, however for the future we also want to include more features and tools to help us with the writing process.
